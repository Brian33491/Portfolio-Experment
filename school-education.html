School Of Education

Modern responsive website for education department

Role UX Designer & Frontend Developer
Timeline 3 Months
Tools Figma, HTML, CSS, JavaScript

Design Process

Following a user-centered design approach, I conducted stakeholder interviews, created user personas, and developed wireframes before moving to high-fidelity prototypes. The design emphasizes accessibility and mobile-first responsiveness.

1

Research

Conducted user interviews and competitive analysis to understand needs and opportunities.

2

Wireframing

Created low-fidelity wireframes to establish information architecture and user flows.

3

Prototyping

Developed interactive prototypes for user testing and feedback collection.

4

Development

Implemented the final design with clean, semantic HTML, CSS, and JavaScript.

Key Features

Responsive Design

Fully responsive layout that works seamlessly on all device sizes from mobile to desktop.

User-Centered

Designed with a focus on user needs, preferences, and accessibility requirements.

Modern UI

Clean, contemporary interface with intuitive navigation and visual hierarchy.

Program Finder

Interactive tool to help students discover educational programs that match their interests.

Faculty Directory

Searchable directory with filtering options to find faculty members by expertise.

Mobile Optimization

Optimized experience for mobile users with touch-friendly interfaces.

Results & Impact

The new website resulted in a 40% increase in prospective student applications and significantly improved user satisfaction scores. The client reported that faculty and current students found the new site much easier to navigate.
